AutoSlideIdea

Markdownベースのシンプルなプレゼンテーション作成ツール

🎯 プロジェクト概要

AutoSlideIdeaは、Markdownベースのシンプルさを追求し、AI支援機能を活用して効率的にプレゼンテーション資料を作成するためのツールセットです。

✨ 主な機能

  • 📝 Markdownでスライド作成
  • 🤖 AI支援によるコンテンツ生成
  • 🎨 4種類のテーマ（base, academic, business, minimal）
  • 📊 Mermaid図表の自動変換
  • 🚀 GitHub Actions自動ビルド
  • 📦 PDF/PowerPoint/HTML出力
